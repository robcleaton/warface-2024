Multi-brand, multi-platform Design System

Sainsbury's and Argos partnered years ago and wanted to start bringing their resources closer together. The goal was to build a centralised design system that could be adopted by each and affiate brands.

Services
  • Recognising different company cultures
  • Multi-brand UI library
  • Multi-platform UI library
  • Headless Global UI library
  • Design Tokens
